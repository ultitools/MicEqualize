How to Get Started

Before Installation

- Ensure you have administrator rights.
- Close any applications that are using your microphone (e.g., Discord, OBS, Audacity).
- Temporarily disable your antivirus if it interferes with the installation.
- Recommended: Create a system restore point.

After Installation

- Restart your computer to apply changes.
- Open MicEqualize from the Start Menu.
- Select your primary microphone from the device list.
- Start applying filters and hear the changes in real-time!
